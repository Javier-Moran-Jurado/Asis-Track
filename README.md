# AsisTrack – Pantalla de justificación de inasistencias (diseño estático)

Pantalla diseñada para que los usuarios puedan justificar una inasistencia adjuntando documentos, firma de respaldo y motivo. Este archivo contiene solo HTML y CSS, sin funcionalidad JavaScript, optimizado para ser importado en Figma o utilizado como referencia visual.

## Criterios de aceptación cumplidos

- Formulario específico para justificación (motivo, archivos adjuntos, firma de respaldo).
- Visualización de la inasistencia a justificar (fecha, persona).
- Integración visual con el flujo de planillas (indicado en el pie de página).
## Características del diseño

- Enfoque mobile-first: elementos táctiles de 44px de altura mínima, disposición en columna en móvil.
- Tarjeta destacada con la inasistencia pendiente (fecha, estudiante, estado).
- Campos de formulario: motivo (select), descripción (textarea), adjuntos (simulación de archivos), firma (área estática con texto de ejemplo).
- Botones de acción: Cancelar y Enviar justificación (sin funcionalidad, solo aspecto visual).
- Estilos basados en la guía de AsisTrack (colores primarios, secundarios, grises, tipografía Inter, sombras, bordes redondeados).
## Diseño responsive

- En pantallas menores a 768px: los elementos se apilan verticalmente, los botones ocupan todo el ancho.
- En pantallas mayores: la tarjeta de inasistencia se expande horizontalmente y los botones se alinean a la derecha.

## Nota

Este diseño es puramente visual, sin lógica de backend ni interactividad real. Es ideal para validar la interfaz con el equipo y como base para un prototipo funcional en Figma.
