# AsisTrack – OCR móvil (previsualización y edición)

Pantalla que simula la extracción de datos desde una imagen mediante OCR, permite editar el texto y corregir palabras con sugerencias.

## Qué cumple

- Imagen original + texto extraído editable  
- Botones: Confirmar / Cancelar  
- Sugerencias para corregir palabras inciertas  
- Diseño responsive (móvil primero)

---

## Componentes

- Área de texto editable  
- Tarjetas de palabras con baja confianza  
- Chips de sugerencias (al hacer clic reemplazan la palabra)  
- Toast de notificaciones  

---

## Vista móvil

- Botones y chips con tamaño táctil (mínimo 44px)  
- Diseño en columna en móvil, dos columnas en tablet/escritorio
